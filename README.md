Симуляция банкомата (ATM Simulator)

Постановка задачи

Разработать консольное приложение на C#, имитирующее работу банкомата. Программа должна обеспечивать следующие возможности:

- Авторизация пользователя по имени и PIN-коду
- Просмотр текущего баланса
- Пополнение счёта
- Снятие денег (с контролем достаточности средств)
- Перевод средств другому пользователю

Применённый паттерн

Singleton

Зачем применяется

Паттерн нужен, чтобы во всей программе существовал только один экземпляр банкомата. Это логично, потому что:

- Банкомат - это реальное устройство, которое не может существовать в нескольких копиях
- Все финансовые операции должны проходить через один и тот же объект
- Исключается ситуация, когда случайно создаётся второй банкомат, что могло бы привести к рассинхронизации данных

Какую проблему решает

Singleton делает конструктор класса закрытым (`private ATM()`) и предоставляет статическое свойство `Instance`, которое всегда отдаёт один и тот же объект. Благодаря этому все данные о пользователях и их балансах хранятся в едином месте.

Требования для запуска

Необходимо: .NET SDK 6.0 или новее
